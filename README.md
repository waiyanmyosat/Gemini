Gemini Native View (Kotlin)
An optimized arm64 web-native wrapper for Android 16, built with Kotlin for a fast, lightweight experience.

The Build
In-App Auth: Log in directly within the app. No annoying redirects to external browsers—keep your session contained and secure.

Smart Link Handling: Only Gemini stays in the app. Any other links automatically open in your default browser to keep your workspace clean.

Auto-Theme Sync: The status bar and UI adapt instantly to your system’s Dark/Light mode settings.

Key Features
Play Services Independent: Works perfectly without Google Play Services or the bloated official Google app. Ideal for "de-Googled" devices or anyone wanting a minimal setup.

Built for Performance: Optimized for arm64 architecture to ensure smooth scrolling and quick response times.

Built for speed and independence.
